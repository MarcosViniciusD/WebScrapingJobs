# coleta de vagas

Este script acessa o site da A Executiva RH, filtra vagas por cidades e salva os resultados em uma planilha Excel chamada `vagas.xlsx`.

## Como usar

1. Instale o Python no computador
    Marque a opção "Add to PATH" ao instalar.

2. Instale as dependências do projeto:
   pip install -r requirements.txt
